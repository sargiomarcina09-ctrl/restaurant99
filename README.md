# Aurelia Table Restaurant Website

Luxury fine dining restaurant website built with Next.js, React, Framer Motion, and Tailwind CSS.

## Correct GitHub Structure

Your GitHub repo root must look like this:

```text
app/
public/
package.json
next.config.ts
postcss.config.mjs
tsconfig.json
vercel.json
README.md
```

Do not put `page.tsx`, `layout.tsx`, `globals.css`, `favicon.svg`, `robots.txt`, or `sitemap.xml` directly in the repo root.

## Vercel Settings

- Framework Preset: Next.js
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: leave blank
